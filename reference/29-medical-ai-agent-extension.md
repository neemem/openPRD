## 2. 医疗 AI Agent 关键合规(8 大要求)

### 2.1 数据来源与训练

| 要求 | 实现 |
|---|---|
| **合法数据源** | 公开论文 + 合作机构 + 用户授权数据 |
| **数据脱敏** | 去除 PHI 标识符 |
| **数据多样性** | 跨地域、跨人群、跨机构 |
| **数据时效** | 知识库定期更新(季度) |
| **数据可追溯** | 训练数据来源、版本、许可 |

### 2.2 模型验证(关键!)

| 维度 | 要求 |
|---|---|
| **准确性** | 准确率 / F1 / 精确率 / 召回率 |
| **鲁棒性** | 抗对抗样本、噪声测试 |
| **公平性** | 跨人群公平(性别/年龄/族裔) |
| **可解释性** | 决策依据可追溯 |
| **可重现性** | 相同输入 → 相同输出(温度=0) |
| **可监控** | 在线监控漂移/性能 |

### 2.3 临床验证

| 阶段 | 内容 |
|---|---|
| **回顾性验证** | 用历史病历测试 |
| **前瞻性验证** | 在真实临床环境测试 |
| **多中心验证** | ≥ 3 家医院 |
| **RWE 验证** | 真实世界数据 |

### 2.4 持续学习监管

**FDA Pre-Cert 模式** vs **传统模式**:

| 维度 | 传统模式 | Pre-Cert 模式 |
|---|---|---|
| 上市前 | 严格审批 | 软件质量评估 |
| 上市后 | 重新审批 | PCCP(预定变更控制) |
| 算法更新 | 不可自动 | 可自动(在 PCCP 内) |
| 监控 | 不要求 | 必须持续 |

### 2.5 数据安全与隐私

| 要求 | 实现 |
|---|---|
| **训练数据脱敏** | 去除 PHI 标识 |
| **推理时数据保护** | 输入 PHI 加密 |
| **输出数据安全** | 不展示 PHI 原文 |
| **审计日志** | 所有问询/响应可追溯 |
| **数据本地化** | 重要数据不出境 |

### 2.6 内容安全(防幻觉)

| 风险 | 缓解 |
|---|---|
| **幻觉** | 知识库 RAG + 引用来源 |
| **错误信息** | 免责声明 + 医师审核 |
| **误导用户** | 明确"AI 辅助,不替代医师" |
| **不当医疗建议** | 关键词过滤 + 风险话术拦截 |

### 2.7 责任与免责

| 角色 | 责任 |
|---|---|
| **AI 提供方** | 模型准确性、幻觉防护、安全 |
| **医疗机构** | 医师最终决策 |
| **医师** | 临床判断 |
| **患者** | 知情同意 |

**关键免责**:
- ✅ AI 仅供参考,不替代医师
- ✅ 不用于急症、首诊
- ✅ 重大决策需医师确认

### 2.8 上市后监测

| 监测项 | 方法 |
|---|---|
| **性能漂移** | 在线 A/B 测试 |
| **错误率** | 用户反馈 + 专家审核 |
| **用户投诉** | 客服 + 反馈系统 |
| **不良事件** | 24 小时内响应 |

## 6. 医疗 AI Agent 知识库架构

### 6.1 知识库 5 大来源

| 来源 | 占比 | 特点 | 风险 |
|---|---|---|---|
| **公开医学文献** | 40% | PubMed/Cochrane/UpToDate | 版权 |
| **临床指南** | 25% | NICE/中华医学会/CDC | 时效性 |
| **医学教科书** | 15% | 默克/西氏/八年制 | 出版时间 |
| **合作机构** | 15% | 三甲医院/学会 | 隐私 |
| **用户授权数据** | 5% | 真实问询 | PHI 严格保护 |

### 6.2 RAG 架构(检索增强生成)

```

│            用户问询(文字/语音)              │

                 ▼

│         Query Understanding                │
│         (问询理解 + 实体抽取)               │
└────────────────┬────────────────────────────┘
                 ▼
┌─────────────────────────────────────────────┐
│       知识库检索(混合检索)                  │
│  - 向量检索(Semantic)                    │
│  - 关键词检索(BM25)                      │
│  - 元数据过滤(科室/语言/时间)            │
└────────────────┬────────────────────────────┘
                 ▼
┌─────────────────────────────────────────────┐
│       Prompt 构建                          │
│  - 系统提示(角色 + 边界)                 │

                 ▼

                 ▼
```
