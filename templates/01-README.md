## 3. 快速开始

### 3.1 环境要求

| 环境 | 版本 | 说明 |
|------|------|------|
| Node.js | 18+ | 前端开发 |
| Java | 17+ | 后端开发（如有） |
| MySQL | 8.0+ | 数据库（如有） |
| Redis | 6.x+ | 缓存（如有） |

### 3.2 安装步骤

```bash
# 1. 克隆代码
git clone [仓库地址]

# 2. 进入目录
cd [项目目录]

# 3. 安装依赖
# 前端
cd web && npm install
# 后端（如有）
cd server && mvn install

# 4. 启动开发环境
# 前端
npm run dev
# 后端（如有）
mvn spring-boot:run
```

## 7. 贡献指南

### 7.1 开发流程

```
1. 创建任务 → 2. 开发实现 → 3. 自测 → 4. Code Review → 5. 合并 → 6. 部署
```

### 7.2 提交规范

| 类型 | 说明 | 示例 |
|------|------|------|
| feat | 新功能 | feat: 添加用户登录 |
| fix | 修复 | fix: 修复分页bug |
| docs | 文档 | docs: 更新README |
| style | 格式 | style: 格式化代码 |
| refactor | 重构 | refactor: 重构用户服务 |
| test | 测试 | test: 添加单元测试 |

### 7.3 分支规范

| 分支 | 用途 |
|------|------|
| main | 主分支（生产） |
| develop | 开发分支 |
| feature/* | 功能分支 |
| hotfix/* | 紧急修复 |
